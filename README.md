# Luxury Housing Analysis

## 📌 Project Overview
This project analyzes luxury housing data to find insights about pricing, location, and buyer trends.

## 🛠 Tools Used
- Python (Pandas, NumPy,matplotlib)
- MySQL
- Power BI

## 📊 Key Insights
- Property prices vary by location
- Larger houses have higher prices
- Buyer types show different trends
- Configuration affects pricing
